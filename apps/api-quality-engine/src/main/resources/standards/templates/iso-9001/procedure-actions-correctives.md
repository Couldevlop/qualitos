# Procédure — Non-conformités & actions correctives

> Modèle ISO 9001:2015 §10.2. S'instancie dans le module CAPA de QualitOS.

## 1. Objet
Réagir aux non-conformités, en éliminer les causes et empêcher la récurrence.

## 2. Traitement de la NC (§10.2.1)
- Maîtriser et corriger la NC ; gérer les conséquences.
- Évaluer le besoin d'agir sur les **causes** (analyse de causes — Ishikawa / 5 Pourquoi).

## 3. Action corrective
- Définir, planifier, mettre en œuvre les actions.
- Déterminer si des NC similaires existent ailleurs.
- Mettre à jour risques/opportunités si nécessaire.

## 4. Vérification d'efficacité
Mesurer l'efficacité de l'action (indicateur, délai), clôturer si concluant.

## 5. Informations documentées (§10.2.2)
Nature des NC, actions menées, résultats des actions correctives.

## 6. Preuves attendues (Standards Hub)
Registre NC/CAPA, analyses de causes, preuves d'efficacité à 3/6/12 mois.
